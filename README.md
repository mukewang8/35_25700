<h3>课程介绍：</h3>
<p>集合框架被设计成要满足以下几个目标。</p>
<p>该框架必须是高性能的。基本集合（动态数组，链表，树，哈希表）的实现也必须是高效的。</p>
<p>该框架允许不同类型的集合，以类似的方式工作，具有高度的互操作性。</p>
<p>对一个集合的扩展和适应必须是简单的。</p>
